Asistencia Universitaria — Gestión Scrum (Taiga)
Este repositorio contiene la documentación y evidencias del proyecto Asistencia Universitaria gestionado con Scrum usando Taiga.io.

Objetivo
Desarrollar un sistema que permita:
- Registrar asistencia por parte de docentes.
- Consultar historial de asistencia por parte de estudiantes.
- Gestionar usuarios y roles (administrador, docente, estudiante, coordinador).

Sprint 1
- Periodo: 06–18 nov 2025
- Meta del Sprint: Entregar un primer incremento funcional con registro, consulta y gestión básica de usuarios.
- Historias incluidas: `#1 Registro de asistencia`, `#2 Consulta de asistencia`, `#4 Gestión de usuarios y roles`.

Incremento entregado
- Registro de asistencia por docente (funcional).
- Consulta de asistencia por estudiante (funcional).
- Gestión de usuarios y roles (básico).
